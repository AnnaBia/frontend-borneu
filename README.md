# Borneu — Frontend

Projeto frontend desenvolvido como parte da Atividade Prática Supervisionada (APS) do curso de Bacharelado em Ciência da Computação. Este repositório contém o frontend para visualização de dados de previsão de desmatamento na ilha de Bornéu, desenvolvido em Angular com PO UI, consumindo o backend disponível em outro repositório.

**Repositório do backend:** https://github.com/AnnaBia/backend-borneu.git

## Tecnologias utilizadas

- **Node.js** 18.19.1 (recomenda-se `nvm` para gerenciar versões)
- **Angular CLI** 19.0.5
- **PO UI** (biblioteca de componentes utilizada quando aplicável)

## Instalação

1. Clone o repositório:
    ```bash
    git clone https://github.com/AnnaBia/frontend-borneu.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd frontend-emergy
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```

## Uso

- Para iniciar o servidor de desenvolvimento:
    ```bash
    ng serve
    ```
    O projeto estará disponível em `http://localhost:4200`.

- Para abrir o navegador automaticamente:
    ```bash
    ng serve --o
    ```
